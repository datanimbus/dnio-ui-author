Default: datanimbus.io.faas.base:{{version}}
  1. Create a new Dockerfile using:
    FROM <image-registry>/datanimbus.io.b2b.base:{{version}}
  2. Add NPM Libraries using:
    RUN npm install <library-name>
  3. Repeat step 2, for all the npm libraries you need
  4. Add CMD ["node", "app.js"]  in the last line
  5. Build the Docker Image using the Dockerfile with a tag of your choice
    docker build -t <your-tag> .
  6. Add the Image Tag in the Left Section
  7. Repeat steps 2-6 for Function Base Image using:
    FROM <image-registry>/datanimbus.io.faas.base:{{version}}
